===> SETUP
 - MySQL Database (v. 5.7) ->  creezi DB si ii pui numele in src/main/resources/application.yml la url: jdbc:mysql://localhost:3306/DB_NAME
 - Java 8
 - Intellij
 - Lombok -> plugin de Intellij: Settings -> Plugins -> search lombok
 - Nodejs (dupa install, "npm install -g @angular/cli")


===> STARTING
 - Server -> din Intellij
 - Client -> intri in terminal, mergi in client folder, "npm start"
